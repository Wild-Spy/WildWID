mx_err_t mList_RemoveItemWithID ( uint32_t  remID)

Remove first item with specified ID from list.

Search through the list for an item with the specified ID. If one is found, remove it from the list. Shuffle all following list entries back one spot to fill in the memory gap. eg. if item 2 was removed and list had 4 items, old list looks like 1,2,3,4. New list has 3 items and looks like 1,3,4. Note that a maximum of one item will be removed from the list and it will be the first occurrence of that ID in the list if the ID appears more than once.

Parameters:
IDID to remove.
Returns:
#mx_err_NotFound or #mx_err_Success
